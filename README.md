A simple converter from R.swift to Asset Catalog for Xcode 15

## Requirement
- macOS 13+
- Xcode 15+
- Swift 5.9+

## Installation
### Mint

```sh
mint install 417-72KI/AssetCatalogConverter
```

### Homebrew

```sh
brew install 417-72KI/tap/asset-catalog-converter
```

## Usage
```sh
$ asset-catalog-converter /path/to/project
```
